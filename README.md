# 🤖 AI Stars — Telegram Bot

Telegram-бот для продажи нейросети по Brawl Stars с красивым Web App интерфейсом.

## ⚡ Возможности

- 🛒 Красивый Web App (Mini App) с тёмной темой и анимациями
- ⭐ Оплата через Telegram Stars (нативная)
- 💳 Оплата в рублях и долларах (ручное подтверждение)
- 📊 Управление подписками (SQLite)
- 👨‍💻 Админ-панель для подтверждения платежей

## 🚀 Установка и запуск

### 1. Установите зависимости

```bash
pip install -r requirements.txt
```

### 2. Настройте бота

Откройте `config.py` и заполните:

```python
BOT_TOKEN = "ваш_токен_от_BotFather"
ADMIN_IDS = [ваш_telegram_id]
WEBAPP_URL = "https://ваш-домен.com/index.html"
```

#### Как получить BOT_TOKEN:
1. Откройте @BotFather в Telegram
2. Отправьте `/newbot`
3. Следуйте инструкциям, скопируйте токен

#### Как узнать свой Telegram ID:
1. Откройте @userinfobot в Telegram
2. Нажмите /start — бот покажет ваш ID

#### Как настроить Web App URL:

**Вариант 1 — GitHub Pages (бесплатно):**
1. Создайте репозиторий на GitHub
2. Загрузите папку `webapp/` в репозиторий
3. Включите GitHub Pages в настройках
4. URL будет: `https://ваш-логин.github.io/репо/index.html`

**Вариант 2 — ngrok (для тестирования):**
```bash
# Установите ngrok: https://ngrok.com/
cd webapp
python -m http.server 8080
# В другом терминале:
ngrok http 8080
```

### 3. Включите платежи (для Telegram Stars)

1. Откройте @BotFather
2. Отправьте `/mybots` → выберите бота → `Payments`
3. Включите Telegram Stars

### 4. Запустите бота

```bash
python bot.py
```

## 🛠 Команды бота

### Для пользователей:
- `/start` — Приветствие + магазин
- `/status` — Проверка подписки

### Для администраторов:
- `/pending` — Список ожидающих платежей
- `/confirm <id>` — Подтвердить платёж

## ❓ FAQ

**Q: Как принимать оплату в рублях автоматически?**
A: Подключите платёжного провайдера (ЮKassa, Робокасса) через @BotFather. После этого можно будет принимать оплату прямо в Telegram.

**Q: Безопасно ли хранить данные в SQLite?**
A: Для небольшого бота — да. При росте пользователей рекомендуется перейти на PostgreSQL.
